# WOD Watcher

Forget about having to refresh your local box's website in anticipation of
tomorrows WOD! 

WOD watcher is designed to periodically check for the latest WOD. When a new WOD
is posted, users will be notified by SMS. 



### Start the app
```$xslt
sbt run 
```

### Run Tests 
```$xslt
sbt test 
```
